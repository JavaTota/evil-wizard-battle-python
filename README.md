# Evil Wizard Battle (Python CLI Game)

A turn-based command-line game built in Python where the player selects a character and battles an Evil Wizard using attacks, abilities, and healing.

This project focuses on object-oriented programming concepts such as inheritance, methods, and class interactions.

---

## Features

- Multiple character classes:
  - Warrior
  - Mage
  - Archer
  - Paladin
- Unique abilities for each class
- Randomized attack damage
- Healing system with max health cap
- Turn-based battle system
- Enemy AI (Evil Wizard regenerates and attacks)
- Victory and defeat conditions
